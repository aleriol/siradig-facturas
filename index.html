📊 Google Sheets — Apps Script

La app se conecta a tu Google Sheet via Apps Script (gratuito, sin OAuth). Seguí estos pasos una sola vez:

  1. Abrí tu Google Sheet: Abrir Sheet ↗
  2. Andá a Extensiones → Apps Script
  3. Borrá todo el código que hay y pegá el código que aparece abajo
  4. Guardá (Ctrl+S), después hacé clic en Implementar → Nueva implementación
  5. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier usuario
  6. Copiá la URL de implementación y pegala acá abajo
📋 Ver código Apps Script (clic para expandir)
